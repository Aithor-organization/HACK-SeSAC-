# Study Quest - 데모 시연 스크립트

**시연 시간**: 5분
**대상**: HACK-SeSAC 본선 심사위원
**목표**: Study Quest의 핵심 가치 전달 및 MVP 시연

## 🎬 시연 구성 (5분)

### 1. 문제 제시 (30초)

**스크립트**:
```
안녕하세요. Study Quest 팀입니다.

여러분, 재수생들이 가장 힘들어하는 게 뭘까요?

(슬라이드: 재수생 설문 조사 결과)

"수능까지 8개월... 뭘 어떻게 해야 할지 막막해요"
"혼자 공부하다 보니 우울하고 불안해요"

전국 18만명의 재수생들이 이런 고민을 하고 있습니다.
학원비는 월 250만원, 연간 3,000만원입니다.
```

**슬라이드**:
- Pain Point 통계
- 학원비 비교 그래프

---

### 2. 솔루션 소개 (1분)

**스크립트**:
```
Study Quest는 이 문제를 AI로 해결합니다.

(슬라이드: Study Quest 로고 + 태그라인)
"막연한 목표를 오늘 할 수 있는 퀘스트로"

3가지 핵심 기능을 제공합니다:

1️⃣ AI 자동 목표 분해 (업계 최초)
   막연한 목표 → 일일 실행 가능한 퀘스트

2️⃣ AI 캐릭터 동반자 (3종)
   계획 수립부터 멘탈 관리까지

3️⃣ 진도 추적 및 알림
   매일 오전 8시, 오늘의 퀘스트 알림

학원비 250만원 → Study Quest 1.5만원
**94% 비용 절감**, 그러면서도 **개인 맞춤형 학습 계획**을 제공합니다.
```

**슬라이드**:
- 3가지 핵심 기능 아이콘
- 비용 비교 (학원 vs Study Quest)

### 3. MVP 시연 (3분)

#### 3-1. 온보딩 플로우 (40초)

**스크립트**:
```
실제 앱을 보여드리겠습니다.

(앱 실행: 스플래시 화면)

앱을 처음 실행하면 5단계 온보딩이 시작됩니다.

(화면 1: 기본 정보)
이름과 학년을 입력합니다.

(화면 2: 현재 성적)
"수학 70점, 영어 80점..."

(화면 3: 목표 설정)
"목표: 수학 90점, 서울대학교"

(화면 4: 수능 D-Day)
자동으로 수능일까지 계산됩니다. "D-240"

(화면 5: AI 캐릭터 선택)
3종 캐릭터 중 하나를 선택합니다.
저는 "엄근진 멘토형"을 선택하겠습니다.
```

**조작**:
- 각 화면 2-3초씩 빠르게 진행
- 미리 입력값 준비 (빠른 시연)

#### 3-2. AI 자동 목표 분해 (50초)

**스크립트**:
```
이제 핵심 기능입니다. AI 자동 목표 분해!

(화면: AI 분해 로딩)
"목표를 분석 중이에요..."

5초 후, AI가 8개월 전체 계획을 자동으로 만들어줍니다.

(화면: 생성된 계획)

Phase 1 (1-3개월): 기본 개념 완성
- 수학I 함수, 수열 집중 학습

Phase 2 (4-6개월): 응용 문제 연습
- 고난도 문제 풀이

Phase 3 (7-8개월): 실전 모의고사
- 기출 모의고사 주 3회

그리고 오늘부터 바로 시작할 수 있는 **일일 퀘스트**가 생성됩니다.
```

**조작**:
- 미리 생성된 결과 사용 (시연 시간 단축)
- 또는 실제 API 호출 (5초 대기)

#### 3-3. 홈 화면 - 일일 퀘스트 (40초)

**스크립트**:
```
(화면: 홈 화면)

홈 화면에는 오늘의 퀘스트 3개가 표시됩니다.

✅ 수학I 함수 개념 정리 (30분)
✅ 수학I 함수 기본 문제 10개 (60분)
✅ 영어 단어 50개 암기 (30분)

체크박스를 누르면...

(퀘스트 1개 완료)

축하 애니메이션과 함께 진도율이 실시간으로 업데이트됩니다.

오늘 진도율: 33% → 66%
주간 진도율: 45% → 52%

차트로 한눈에 확인할 수 있습니다.
```

**조작**:
- 퀘스트 1개 체크
- 애니메이션 재생
- 진도율 차트 스크롤

#### 3-4. AI 캐릭터 대화 (50초)

**스크립트**:

마지막으로, AI 캐릭터와의 대화입니다.

사용자가 질문하면...

3초 이내에 응답이 옵니다.

캐릭터는 언제든 변경 가능합니다.

(캐릭터 변경: 공감 친구형)

"어제 못한 거 괜찮아. 오늘은 영어만 30분 해보자."

## 🎯 예상 질문 & 답변

### Q1: "AI 응답이 정확하지 않으면 어떻게 하나요?"

**답변**:
```
좋은 질문입니다.

1. 사용자 피드백 시스템
   - AI 계획에 "도움됨/안됨" 평가 버튼
   - 평가 데이터로 Prompt 개선

2. 수정 가능
   - 생성된 계획은 사용자가 직접 수정 가능
   - 난이도 조절 (쉽게/보통/어렵게)

3. 지속 학습
   - 베타 테스터 100명 데이터로 정확도 향상
   - 현재 목표: 90% 정확도
```

### Q2: "GPT-4o API 비용이 너무 높지 않나요?"

**답변**:
```
비용 최적화 전략이 있습니다.

1. Redis 캐싱
   - 동일 목표는 24시간 캐싱
   - 약 30% 비용 절감

2. 무료 사용자 제한
   - 하루 10회 대화 제한
   - 프리미엄은 무제한

3. 실제 비용
   - 사용자당 월 470원
   - 월 1,000명 기준: 47만원
   - 충분히 감당 가능한 수준입니다.

향후 자체 AI 모델 개발도 검토 중입니다.
```

### Q3: "기존 학습 앱과의 차이점은?"

**답변**:
```
기존 앱과의 핵심 차이는 2가지입니다.

1. AI 자동 목표 분해
   - 기존 앱: 사용자가 직접 계획 작성
   - Study Quest: AI가 자동으로 분해 (5초)

2. 멘탈 관리 통합
   - 기존 앱: 타이머, 기록만
   - Study Quest: AI 캐릭터가 멘탈 케어

학습 "계획"과 "멘탈 관리"를 하나의 앱에서 제공하는 건
Study Quest가 유일합니다.
```

### Q4: "재수생만 타겟인가요?"

**답변**:
```
1차 타겟은 재수생이지만, 확장 계획이 있습니다.

Year 1: 재수생 18만명
Year 2: 고3 수험생 40만명 진출
Year 3: 전체 고등학생 130만명

장기적으로는:
- 취업 준비생 (공무원, 자격증)
- 직장인 자기계발
- 평생 학습 플랫폼으로 확장

"목표 달성"이라는 보편적 니즈를 다룹니다.
```

### Q5: "실제 효과가 검증되었나요?"

**답변**:
```
현재 베타 테스트 준비 중입니다.

계획:
- 2026년 2월: 베타 테스터 100명 모집
- 2개월 사용 테스트
- 목표 지표:
  * NPS 50+
  * D7 재방문율 50%
  * 만족도 4.0/5.0

유사 서비스 사례:
- Duolingo (언어 학습): AI 기반 일일 퀘스트로 D7 재방문율 50%
- Forest (집중 타이머): 게이미피케이션으로 MAU 1,000만명

Study Quest는 이 두 가지를 결합했습니다.
```
