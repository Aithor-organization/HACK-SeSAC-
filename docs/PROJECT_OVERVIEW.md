# Study Quest - 프로젝트 개요

**프로젝트명**: Study Quest
**버전**: MVP 1.0
**작성일**: 2025-11-17
**해커톤**: HACK-SeSAC

---

## 🎯 핵심 아이디어

재수생/N수생의 **막연한 목표**를 AI가 **실행 가능한 일일 퀘스트**로 자동 분해하고,
**3종 AI 캐릭터 동반자**가 계획 수립부터 멘탈 관리까지 제공

---

## 💡 문제 정의

### 타겟 사용자
- **1차 타겟**: 재수생/N수생 18만명 (2025년)
- **2차 타겟**: 고3 수험생 40만명
- **연령**: 18-19세
- **특징**: 하루 10-12시간 학습, 스마트폰 활용도 높음

### Pain Points

1. **계획 수립의 어려움**
   - "수능까지 8개월, 뭘 어떻게 해야 할지 막막해요"
   - 현재: Excel 수기 작성 또는 학원 플래너
   - 문제: 장기 목표를 일일 계획으로 분해하기 어려움

2. **중도 포기 위험**
   - "처음엔 의지가 강했는데 3개월 지나니 슬럼프가 와요"
   - 현재: 혼자 극복해야 함
   - 문제: 멘탈 관리 시스템 부재

3. **멘탈 관리 한계**
   - "혼자 공부하다 보니 우울하고 불안해요"
   - 현재: 친구/가족에게 의지
   - 문제: 전문적 멘탈 케어 부재

4. **높은 학원비**
   - 학원비 월 250만원
   - 연간 3,000만원
   - 문제: 경제적 부담

## 🚀 솔루션

### 핵심 기능 3가지

#### 1. AI 자동 목표 분해 ⭐ **업계 최초**

**Before**:
```
사용자: "수학 70점을 90점으로 올리고 싶은데... 어떻게 해야 하지?"
```

**After (AI 자동 분해)**:
```
Phase 1 (1-3개월): 기본 개념 완성
- 수학I 함수: 개념 정리 30분 + 기본 문제 10개
- 수학II 미분: 개념 정리 40분 + 기본 문제 15개

Phase 2 (4-6개월): 응용 문제 연습
- 수학I 함수 응용: 중급 문제 20개
- 수학II 적분: 고급 문제 15개

Phase 3 (7-8개월): 실전 모의고사
- 기출 모의고사 주 3회
- 취약 단원 집중 보강

오늘의 퀘스트:
✅ 수학I 함수 개념 정리 (30분)
✅ 수학I 함수 기본 문제 10개 (60분)
✅ 영어 단어 50개 암기 (30분)
```

**기술**:
- OpenAI GPT-4o API
- 재수생 특화 커리큘럼 데이터 5,000개 학습 주제
- Few-shot Learning 기반 Prompt

#### 2. AI 캐릭터 동반자 (3종)

사용자 성향에 맞는 AI 캐릭터 선택 가능:

| 캐릭터 | 페르소나 | 강점 | 예시 대사 |
|--------|---------|------|----------|
| **A: 엄근진 멘토형** | 30대 입시 컨설턴트 | 계획 수립, 냉정한 피드백 | "지금 진도로는 목표 못 채워. 하루 3문제 더 풀어." |
| **B: 공감 친구형** | 20대 재수 성공 선배 | 멘탈 관리, 격려 | "어제 못한 거 괜찮아. 오늘은 영어만 30분 해보자." |
| **C: 하이브리드형** | 25세 대학생 멘토 | 계획 + 공감 균형 | "오늘 계획: 수학 3시간. 힘들면 쉬어도 돼." |

#### 3. 일일 퀘스트 + 진도 추적

**홈 화면**:
- 오늘의 퀘스트 3-5개 (AI 자동 생성)
- 체크박스 완료 → 즉시 피드백
- 진도율 시각화 (일일/주간/전체)

## 📊 시장 기회

### 시장 규모 (TAM/SAM/SOM)

- **TAM** (전체 고등학생): 130만명 × 1.5만원 = **1,950억원**
- **SAM** (재수생 + 고3): 58만명 × 1.5만원 = **870억원**
- **SOM** (1년 차 목표): 1만명 × 1.5만원 × 12개월 = **1.8억원**

### 성장 전략

**Year 1** (2026년):
- MAU 10,000명
- ARR 1.12억원
- 재수생 커뮤니티 인지도 1위

**Year 2** (2027년):
- MAU 50,000명 (고3 진출)
- ARR 7억원

**Year 3** (2028년):
- MAU 100,000명

- 평생 학습 플랫폼 확장

## 🛠️ MVP 구현 범위 (본선 시연)

### Must Have (필수 구현)

1. ✅ **온보딩** (5단계)
   - 기본 정보 입력
   - 현재 성적 입력
   - 목표 설정
   - AI 캐릭터 선택

2. ✅ **AI 목표 분해**
   - 자연어 목표 입력
   - GPT-4o 자동 분해
   - Phase별 계획 + 일일 퀘스트

3. ✅ **홈 화면**
   - 오늘의 퀘스트 3개
   - 체크박스 완료
   - 진도율 표시

4. ✅ **AI 캐릭터 대화**
   - 간단한 채팅 UI
   - 캐릭터별 응답

5. ✅ **인증**
   - Guest 모드 (MVP)
   - 또는 간단한 로그인

### Nice to Have (시간 여유 시)

- 푸시 알림 시뮬레이션
- 진도 추적 차트
- 프로필 관리

## 📅 개발 일정

### 예선 준비 (현재 ~ 예선 마감)
- Week 1: 문서 작성 (PRD, 아키텍처, API 명세)
- Week 2: 디자인 완성 (와이어프레임 12개)
- Week 3: 초기 코드 작성 (프로젝트 셋업)

### 본선 준비 (예선 통과 ~ 본선)
- Week 1-2: MVP 핵심 기능 구현 (온보딩, AI 분해, 홈)
- Week 3: AI 캐릭터 통합 및 테스트
- Week 4: 통합 테스트 및 시연 준비
- Week 5: 피칭 자료 및 데모 스크립트

## 🏆 기대 효과

### 사용자 측면
- 명확한 일일 학습 계획 → **계획 수립 시간 90% 절감**
- AI 동반자 → **슬럼프 극복률 60% 향상**
- 저렴한 비용 → **학원비 대비 94% 절감** (월 250만원 → 1.5만원)

### 비즈니스 측면
- 1년 차 ARR 1.12억원
- 14개월 차 BEP 달성
- LTV:CAC 비율 2.4:1 (건강한 수익 구조)

### 사회적 영향
- 경제적 약자 학생들에게 양질의 학습 지원
- AI 기술의 교육 격차 해소
- 재수생 멘탈 헬스 개선
