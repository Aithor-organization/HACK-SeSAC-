## 1. 제품 개요

### 1.1 제품명
**Study Quest** (가칭)

### 1.2 제품 설명
재수생/N수생의 막연한 목표를 AI가 실행 가능한 일일 퀘스트로 자동 분해하고, 다양한 성격의 AI 캐릭터 동반자가 계획 수립부터 멘탈 관리까지 제공하는 목표 달성 학습 플랫폼

### 1.3 핵심 가치 제안
> **"막연한 목표를 오늘 할 수 있는 퀘스트로"**

- ✅ **AI 자동 목표 분해**: 업계 최초, 장기 목표를 일일 실행 가능한 퀘스트로 자동 변환
- ✅ **재수생 특화 AI 캐릭터 3종**: 멘토형/친구형/하이브리드형 선택 가능
- ✅ **계획 + 멘탈 관리 통합**: 학습 계획과 심리적 지원을 하나의 플랫폼에서

### 1.4 타겟 시장
- **1차 타겟**: 재수생/N수생 18만명 (2025년)
- **2차 타겟**: 고3 수험생 40만명 (1년 차 목표)
- **3차 타겟**: 전체 고등학생 130만명 (2-3년 차)

## 2. 제품 비전 및 목표

### 2.1 비전
> **"모든 학생의 목표 달성을 돕는 AI 동반자 플랫폼"**

### 2.2 미션
막연한 장기 목표로 인한 좌절과 실패를 없애고, AI 기반 체계적 계획과 멘탈 케어를 통해 모든 학생이 자신의 목표를 달성할 수 있도록 지원

### 2.3 제품 목표

#### 단기 목표 (1년 차, 2026년)
- MAU 10,000명 달성
- NPS 50+ 달성
- 재수생 커뮤니티 내 인지도 1위
- 프리미엄 전환율 5% 달성

#### 중기 목표 (2-3년 차)
- MAU 100,000명 달성
- 고등학생 시장 진출
- B2B 학원 라이선스 50곳 계약
- 재수생 필수 앱으로 자리매김

#### 장기 목표 (5년+)
- 평생 학습 플랫폼으로 확장 (취업, 자격증, 직장인)
- 글로벌 시장 진출
- 자체 AI 모델 개발

## 3. 사용자 페르소나

### 3.1 Primary Persona: "재수생 김수험 (19세)"

**인구통계학적 정보**
- 나이: 19세 (남성/여성 구분 없음)
- 직업: 재수생 (학원 등록 또는 독학)
- 거주지: 서울/수도권 또는 지방 중소도시
- 경제 상태: 중산층 가정 (부모님 지원)

**행동 패턴**
- 하루 10-12시간 학습 목표
- 스마트폰 사용: 하루 평균 3-4시간 (학습 관리, SNS, 유튜브)
- 주로 사용하는 앱: 유튜브 (강의), 네이버 카페 (오르비, 수만휘), Forest (타이머)
- 학습 관리: Excel 수기 작성 또는 학원 제공 플래너

**Pain Points**
1. **계획 수립의 어려움**: "수능까지 8개월, 뭘 어떻게 해야 할지 막막해요"
2. **중도 포기 위험**: "처음엔 의지가 강했는데 3개월 지나니 슬럼프가 와요"
3. **멘탈 관리 한계**: "혼자 공부하다 보니 우울하고 불안해요"
4. **비싼 학원비**: "학원비 월 250만원은 부담스러워요"

**목표 및 동기**
- 주요 목표: 수능 점수 향상, 목표 대학 합격
- 동기: 작년 실패 만회, 부모님 기대에 부응, 자기 증명

**기술 수용도**
- 스마트폰 활용: 상급 (앱 설치 및 사용에 익숙)
- AI 이해도: 중급 (ChatGPT 사용 경험 있음)
- 새로운 기술 수용: 높음 (도움이 된다면 기꺼이 시도)

**Study Quest 사용 시나리오**
- **사용 빈도**: 하루 3-5회 (아침/점심/저녁/자기 전)
- **주요 사용 상황**:
  - 아침: 오늘의 퀘스트 확인
  - 점심/저녁: 학습 기록 및 진도 체크
  - 자기 전: AI 캐릭터와 대화 (멘탈 관리)
- **기대 효과**: 명확한 일일 학습 계획, 슬럼프 극복, 지속적 동기 부여

**Quote**
> "혼자 공부하면 '내가 잘하고 있는 건가?' 불안해요. 누군가 매일 내 계획을 짜주고, 힘들 때 응원해주면 좋겠어요."

## 4. 핵심 기능 요구사항

### 4.1 기능 분류

기능은 다음과 같이 분류됩니다:
- **P0 (Critical)**: MVP 필수 기능, 없으면 제품 가치 없음
- **P1 (High)**: MVP 출시 후 1-2개월 내 필수
- **P2 (Medium)**: 3-6개월 내 추가
- **P3 (Low)**: 1년 차 이후 고려

---

### 4.2 P0 (Critical) - MVP 필수 기능

#### F1: 사용자 온보딩 및 목표 설정

**User Story**
> **As a** 재수생  
> **I want to** 앱 최초 실행 시 나의 현재 상태와 목표를 입력하고 싶다  
> **So that** AI가 나에게 맞는 학습 계획을 만들어줄 수 있다

**Acceptance Criteria**
- [ ] 사용자는 5단계 온보딩을 완료할 수 있다
  - Step 1: 기본 정보 (이름, 학년)
  - Step 2: 현재 수능 예상 점수 (과목별)
  - Step 3: 목표 점수 및 목표 대학
  - Step 4: 수능 D-Day 설정
  - Step 5: AI 캐릭터 선택 (3종 중 1개)
- [ ] 온보딩 완료 시간: 5분 이내
- [ ] 모든 입력 항목은 선택적이며, 나중에 수정 가능
- [ ] 온보딩 Skip 가능 (기본 설정으로 진행)

**Notes**
- 온보딩 이탈률 목표: 30% 이하
- 사용자가 부담을 느끼지 않도록 간결하게 구성

---

#### F2: AI 기반 자동 목표 분해 (핵심 차별화 기능)

**User Story**
> **As a** 재수생  
> **I want to** "수능까지 수학 70점 → 90점"처럼 막연한 목표를 입력하면  
> **So that** AI가 이를 일일 실행 가능한 퀘스트로 자동 분해해주길 원한다

**Acceptance Criteria**
- [ ] 사용자는 과목별 목표를 자연어로 입력할 수 있다
  - 예: "수학 70점 → 90점", "영어 2등급 → 1등급"
- [ ] AI는 5초 이내에 다음을 생성한다:
  - Phase별 학습 계획 (1-3개월 / 4-6개월 / 7-8개월)
  - 일일 퀘스트 (오늘부터 시작)
- [ ] 생성된 계획은 다음을 포함한다:
  - 학습 주제 (예: "수학I 함수 개념")
  - 예상 소요 시간 (분 단위)
  - 구체적 학습량 (예: "문제 10개")
- [ ] 사용자는 생성된 계획을 수정할 수 있다
  - 난이도 조절 (쉽게 / 보통 / 어렵게)
  - 하루 학습 시간 조절 (4-12시간)

**Technical Notes**
- AI 모델: GPT-4o API (초기), 이후 Claude Sonnet 4.5 A/B 테스트
- Prompt Engineering: 재수생 특화 커리큘럼 데이터 학습
- 정확도 목표: 90% 이상 (사용자 만족도 기준)

**Dependencies**
- AI API 통합 완료
- 과목별 커리큘럼 데이터베이스 구축

#### F3: AI 캐릭터 동반자 (3종)

**User Story**
> **As a** 재수생  
> **I want to** 나의 성향에 맞는 AI 캐릭터를 선택하고  
> **So that** 학습 계획과 멘탈 관리를 받고 싶다

**Acceptance Criteria**
- [ ] 3종의 AI 캐릭터 제공:
  - **캐릭터 A: "엄근진 멘토형"**
    - 페르소나: 30대 입시 컨설턴트, 데이터 기반 조언
    - 강점: 계획 수립, 진도 체크, 냉정한 피드백
    - 예시 대사: "지금 진도로는 목표 못 채워. 하루 3문제 더 풀어."
  - **캐릭터 B: "공감 친구형"**
    - 페르소나: 20대 재수 성공 선배, 공감 능력
    - 강점: 멘탈 관리, 격려, 작은 성취 축하
    - 예시 대사: "어제 못한 거 괜찮아. 오늘은 영어만 30분 해보자."
  - **캐릭터 C: "하이브리드형"**
    - 페르소나: 25세 대학생 멘토, 친근+단호
    - 강점: 계획 + 공감 균형, 상황별 모드 전환
    - 예시 대사: "오늘 계획: 수학 3시간. 힘들면 쉬어도 돼. 시작해볼까?"
- [ ] 사용자는 캐릭터를 언제든 변경 가능
- [ ] 캐릭터와 텍스트 기반 대화 가능
  - 응답 속도: 3초 이내
  - 대화 히스토리 유지 (최근 30일)
- [ ] 무료 사용자: 하루 10회 대화 제한
- [ ] 프리미엄 사용자: 무제한 대화

**Technical Notes**
- System Prompt 설계: 각 캐릭터별 페르소나 정의
- 대화 컨텍스트 관리: 최근 10턴 유지
- 감정 분석: 사용자 메시지의 감정 상태 감지 (슬럼프 징후)

#### F4: 일일 퀘스트 제공 및 진도 추적

**User Story**
> **As a** 재수생  
> **I want to** 매일 앱을 열면 "오늘 할 일"이 명확하게 보이고  
> **So that** 고민 없이 바로 학습을 시작할 수 있다

**Acceptance Criteria**
- [ ] 홈 화면에 "오늘의 퀘스트 3-5개" 표시
  - 예: "수학I 함수 문제 10개", "영어 단어 50개 암기"
- [ ] 각 퀘스트는 체크박스로 완료 표시 가능
- [ ] 완료한 퀘스트는 즉시 시각적 피드백 (애니메이션, 사운드)
- [ ] 진도율 표시:
  - 일일 진도율 (오늘의 퀘스트 완료도)
  - 주간 진도율 (이번 주 목표 대비)
  - 전체 진도율 (수능까지 남은 기간 대비)
- [ ] 진도율은 차트로 시각화 (막대 그래프, 선 그래프)

**MVP 제외 항목 (P1으로 연기)**
- 타이머 기능
- 학습 시간 자동 추적
- 과목별 상세 통계

### 4.3 P1 (High) - MVP 출시 후 1-2개월 내 추가

#### F7: 슬럼프 감지 및 자동 개입

**User Story**
> **As a** 재수생  
> **I want to** 내가 슬럼프에 빠졌을 때 AI가 알아채고  
> **So that** 중도 포기하지 않도록 도움을 받고 싶다

**Acceptance Criteria**
- [ ] 슬럼프 감지 알고리즘:
  - 연속 3일 계획 미달성 → "슬럼프 위험" 경고
  - 주간 진도율 -20% 이상 하락 → "긴급 개입"
  - 대화 내 부정적 감정 키워드 감지 → "멘탈 케어 필요"
- [ ] 자동 개입 액션:
  - AI 캐릭터 자동 전환 (멘토형 → 친구형)
  - 목표 난이도 임시 하향 조정
  - "작은 승리" 퀘스트 제공 (30분만 공부하기)
  - 격려 메시지 자동 발송
- [ ] 슬럼프 극복 가이드 제공 (아티클/영상)

**Success Metrics**
- 슬럼프 감지 정확도: 80% 이상
